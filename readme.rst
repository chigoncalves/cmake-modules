###############
 cmake-modules
###############

A tiny collection of `cmake <http://cmake.org>`_ modules.

*********
 Modules
*********

* ``FindGtk3``
       finds Gtk+3.

* ``FindGuile``
       finds Guile 2.0.

* ``FindCUnit``
       finds CUnit.

* ``FindRSVG``
       finds RSVG.

* ``FindSN``
       finds startup notification.

* ``FindImlib``
       finds Imlib.

* ``FindGC``
       finds Boehm garbage collector.
